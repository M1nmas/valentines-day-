# Valentine's Day Catch Game ❤️

A 3D web game built with React Three Fiber just for Valentine's Day!

## How to Play
- **Objective:** Catch as many hearts as you can before time runs out!
- **Controls:** Click on the floating hearts to collect them.
- **Duration:** You have 30 seconds.

## Tech Stack
- **Framework:** React (Vite)
- **3D Engine:** React Three Fiber (Three.js)
- **Styling:** CSS

## Setup & Run
1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the development server:
   ```bash
   npm run dev
   ```
3. Open your browser at `http://localhost:5173`

## Features
- Interactive 3D Hearts
- Score tracking
- Timer system
- Beautiful background with stars
- Responsive design
